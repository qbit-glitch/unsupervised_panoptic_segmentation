# DCFA and SIMCF-ABC: High-Level Descriptions

## DCFA (Depth-Conditioned Feature Adapter)

DCFA is a tiny neural network (~40K parameters) that sits between a frozen vision backbone and k-means clustering to make semantic pseudo-labels aware of 3D geometry. The problem it solves is that frozen DINOv3 features cluster patches based purely on visual appearance, completely blind to depth — so a sidewalk patch at 5 meters and a road patch at 50 meters can end up in the same cluster because they share similar texture. DCFA takes each 90-dimensional patch feature and its corresponding monocular depth value, encodes the depth into a 16-dimensional sinusoidal signal, and feeds both through a small MLP that outputs a subtle adjustment to the original feature. The network is trained with a depth-correlation loss that pulls together features at similar depths and pushes apart features at different depths, plus a strong preservation term that prevents the adapted features from drifting far from the originals. After adaptation, k-means re-clustering produces semantic labels with sharper boundaries at depth discontinuities, raising mIoU by +2.6 points.

## SIMCF-ABC (Semantic-Instance Mutual Consistency Filtering)

SIMCF-ABC is a three-step post-processing pipeline that cleans up inconsistencies between semantic labels, instance masks, and depth statistics after they have been generated independently. The core observation is that semantics come from feature clustering, instances come from depth-gradient splitting, and these two modalities contain complementary information but also complementary errors — so they can cross-check each other. **Step A** enforces that every instance region has a single consistent semantic class via majority vote, removing stuff pixels that leaked into thing regions. **Step B** merges over-fragmented instances by checking whether adjacent same-class fragments share similar DINOv3 appearance (cosine similarity > 0.85), which is critical because raw DepthPro splitting produces ~44 fragments per image with 50% stuff contamination, and merging cuts this to ~22 coherent instances with 28% contamination. **Step C** masks semantic labels that are geometrically implausible — for example, a "sky" pixel at 2 meters depth — using per-class global depth statistics. Together these three steps raise pseudo-label PQ by +0.73, and when combined with DCFA they compose near-additively to +1.31 PQ.
