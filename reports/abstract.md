# Unsupervised Panoptic Segmentation from Monocular Images via Feature Overclustering and Depth-Guided Instance Decomposition

---

## Abstract

Unsupervised panoptic segmentation — assigning every pixel a semantic class and instance identity without ground-truth annotations — remains a largely open problem. The current state of the art, CUPS (CVPR 2025), achieves PQ 27.8 on Cityscapes but depends on stereo video sequences with optical flow and binocular disparity. We present a monocular-only pipeline that requires neither stereo pairs nor temporal sequences — only single RGB images and self-supervised pretrained models. Our approach introduces three contributions. First, we propose *overclustered feature semantics*: applying K-means with moderate cluster counts ($k$=80) to the learned 90-dimensional features of a self-supervised semantic segmentation model, recovering seven Cityscapes classes that are entirely absent under the model's default centroid assignment and improving semantic mIoU from 42.9% to 60.7%. Second, we introduce *depth-guided instance decomposition*, which segments thing-class regions into individual objects by detecting depth discontinuities from a self-supervised monocular depth estimator, achieving 6.2$\times$ higher thing-class panoptic quality than spectral graph methods. Third, through a systematic multi-resolution analysis spanning 64 configurations, we identify a previously undocumented tradeoff between stuff segmentation quality and thing instance separability as a function of cluster granularity, establishing that moderate overclustering ($k$=80) achieves the Pareto-optimal balance. On Cityscapes validation, our pseudo-labels achieve PQ 26.74 — within 1.06 points of CUPS's *final trained model* — while using strictly less input (monocular vs. stereo video). Our thing-class quality (PQ$^\text{Th}$=19.41) *surpasses* CUPS (17.70) by 1.71 points, and our segment quality (SQ=71.88) exceeds CUPS (57.40) by 14.5 points, demonstrating that monocular geometric priors yield more precisely delineated instances than multi-frame temporal aggregation. Training a Cascade Mask R-CNN on these pseudo-labels produces a detector reaching PQ 23.8 in an initial run without self-training, with the full pipeline targeting PQ $\geq$ 28.
